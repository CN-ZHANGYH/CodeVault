# 区块链学习路线

# 技术栈图

# 目标

  1. 比特币发展历史，挖矿以及相关应用
  2. 区块链底层技术，如共识算法，侧链技术，密码学知识等，理解PKI基本知识和原理
  3. 研究以太坊技术，了解ERC20,能够自行开发DAPP
  4. 熟悉Hyperledger Fabric 1.0架构，了解其业务流程和设计理念， 相关应用
  5. 了解比特币、以太坊，Ripple，Corda, IOTA， Skycoin等业界热门的区块链设计特点及相关应用

# 方向

目前看来Web3.0的技术，大部分是嘘头。
